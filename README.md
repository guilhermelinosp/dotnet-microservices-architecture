## How to run the project

- Clone this repository

```shell
git clone https://github.com/guilhermelinosp/.git
```
</br>

- Start the Container

```shell
docker compose up --build
```
</br>

- Stop the Container
	- To stop the Container click CTRL+C in your terminal and then type

```shell
docker compose down
```
